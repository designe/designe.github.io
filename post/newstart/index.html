New Start

July 23, 2017
static site generator hugo 새로시작 github git

블로그를 사용할때는 항상 개인 서버를 사용해서 데이터를 관리하곤 했었다. 이 점이 불편하여 이제는 나도 static site generator 를 이용해 블로그를 관리하려 한다.

Open Source가 널리 퍼뜨려지고, Github이 성장하면서 Static site generator 플랫폼도 함께 커져가고 있다.

앞으로 얼마나 클지는 잘 모르겠지만 이 개념을 잘 활용하면 개도국에도 꽤나 요긴하게 활용될 수 있을 것 같다. 물론 아직까지는 좀 접근 장벽이 높아보이긴 하다.
